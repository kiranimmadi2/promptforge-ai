# Universal AI Agent Rules
1. **Understand First**: Analyze codebase before writing code.
2. **Plan Before Editing**: Formulate an explicit plan (e.g., in task.md).
3. **Safe Execution**: Make incremental, testable edits. Avoid large rewrites.
4. **Verify Everything**: Run tests/build steps to verify changes.
5. **Output Format**: Provide concise, structured summaries of modifications.
